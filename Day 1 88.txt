            Track and Manage the source code in git 

       Push Code from Local Repo  --> Remote Repo(https://github.com/gsatkani/demo-day1-88-source-code.git) 

       1.git init   /  git status
       2.git add .  /  git status
       
       3.git remote add origin https://github.com/gsatkani/demo-day1-88-source-code.git

       4.git commit -m "login module completed"   (-m --> Message)
       
       5.git push -u origin master  (-u -> upstream)
